# Certification badges

Drop each certification badge image here using the exact file name referenced
in `src/data/certs.js`, for example:

```
public/certs/oscp.png
public/certs/oswe.png
public/certs/osce3.png
...
```

PNG or SVG both work. Square or transparent badges look best. Until a file
exists, the About page shows a clean placeholder tile in its place — so the
layout always looks finished.
